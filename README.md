# FinTracker - Трекер расходов

Простое и удобное приложение для учёта личных расходов с поддержкой сканирования штрихкодов. Работает как PWA — можно установить на телефон как полноценное приложение.

## Возможности

- Учёт расходов по категориям (еда, транспорт, развлечения, дом, здоровье, подарки, одежда, другое)
- Сканер штрихкодов через камеру телефона
- База товаров с ценами и названиями
- Регулярные платежи с автоматическим списанием
- Статистика за неделю (всего, дней, среднее)
- Корзина покупок
- Поддержка русского и английского языков
- Выбор валюты (рубль, доллар, евро, тенге, гривна, фунт, йена, юань)
- Все данные сохраняются в памяти телефона (localStorage)

## Как установить на телефон

1. Откройте сайт в Chrome на телефоне: `https://frongee521.github.io/FinTracker/`
2. Нажмите три точки в правом верхнем углу
3. Выберите "Добавить на главный экран" или "Установить приложение"
4. Готово — трекер появится на рабочем столе как отдельное приложение

## Как использовать

### Главная
- Выберите категорию, введите сумму и нажмите "Добавить трату"
- Все расходы отображаются в списке ниже

### Сканер
- Нажмите "Запустить" и наведите камеру на штрихкод
- Если товар есть в базе — цена подставится автоматически
- Если товара нет — можно добавить его в базу

### Регулярные
- Добавьте повторяющиеся платежи с указанием дня месяца
- Нажмите "Оплатить" когда пришло время

### Товары
- Просматривайте, редактируйте и удаляйте товары из базы
- Добавляйте новые товары вручную

## Языки

Приложение автоматически определяет язык браузера. Поддерживаются:
- Русский
- Английский

## Технологии

- HTML, CSS, JavaScript
- html5-qrcode для сканера штрихкодов
- PWA (Progressive Web App)
- Service Worker для офлайн-доступа
- localStorage для хранения данных
