# MoneyKo — Финансовый трекер

Простое, быстрое и бесплатное приложение для учёта личных финансов. Работает как PWA — устанавливается на телефон как нативное приложение.

## Возможности

### Основные
- **Учёт расходов** по категориям: еда, транспорт, развлечения, дом, здоровье, подарки, одежда, другое
- **Учёт доходов** с указанием источника
- **Быстрые кнопки** — настраиваемые действия в один клик (проезд, кофе и любые свои)
- **Сканер штрихкодов** — добавляет товары в корзину через камеру
- **Регулярные платежи** — повторяющиеся траты с привязкой к дню месяца

### Продвинутые
- **Месячный бюджет** — прогресс-бар расходов за месяц
- **Лимиты по категориям** — контроль трат по каждой категории
- **Цели накопления** — создавай цели и откладывай на них деньги
- **База товаров** — сохраняй товары со штрихкодами и ценами для быстрого сканирования
- **Поиск похожих штрихкодов** — если товар не найден точно, предлагает похожий (7+ совпадающих цифр)

### Интерфейс
- **Тёмная тема** — светлая, тёмная или автоматическая (как в системе)
- **Два языка** — русский и английский
- **Двухуровневая навигация** — основные табы и выпадающее меню для дополнительных
- **Стильные диалоги** — все уведомления, подтверждения и ввод через кастомные окна

### Синхронизация
- **QR-код** — перенос всех данных между устройствами без интернета и регистрации
- Создаёшь QR на одном устройстве → сканируешь на другом → данные перенесены

### Безопасность
- **Полностью офлайн** — все данные хранятся только на устройстве (localStorage)
- **Без серверов** — никакие данные не отправляются в интернет
- **Без регистрации** — не нужны аккаунты, пароли, токены

## Установка на телефон

1. Открой сайт в Chrome на телефоне: `https://frongee521.github.io/MoneyKo/`
2. Нажми три точки (⋮) в правом верхнем углу
3. Выбери **"Добавить на главный экран"** или **"Установить приложение"**
4. Готово — MoneyKo появится на рабочем столе как отдельное приложение

## Как пользоваться

### Быстрый старт
1. **Добавь доход** — вкладка «Доходы», укажи сумму и источник
2. **Записывай траты** — на главной выбери категорию, введи сумму, нажми «Добавить»
3. **Настрой быстрые кнопки** — вкладка «Регулярные» → «+», укажи название, цену и категорию
4. **Сканируй товары** — вкладка «Сканер» → наведи камеру на штрихкод

### Лимиты и цели
- **Лимиты** (••• → Лимиты) — установи месячный бюджет и лимиты по категориям
- **Цели** (••• → Цели) — создай цель, укажи сумму, откладывай деньги кнопкой «Отложить»

### Синхронизация
- **Шестерёнка** (⚙ внизу справа) → «Создать QR» — покажи код другому устройству
- На втором устройстве: «Сканировать QR» → наведи камеру → готово

## Технологии

- HTML, CSS, JavaScript (без фреймворков)
- html5-qrcode — сканер штрихкодов и QR
- PWA (Progressive Web App) — установка на телефон
- Service Worker — офлайн-доступ
- localStorage — хранение данных
